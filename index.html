Yelp Reviews and Ratings Plot

(The data displayed in this plot has been obtained from Yelp Api )

Wouldn't it be great to have a plot which shows us the Yelp ratings and the number of user reviews of a place? This interactive visualization does exactly that! It illustrates the relationship between the number of reviews and ratings, to help us discern if there is a correlation between the two variables. It also helps find highly rated and most reviewed businesses in a category. Select your options and click on Visualize Data
